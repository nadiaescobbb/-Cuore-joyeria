# Joyería Cuore — Custodios de lo que permanece

Digital editorial platform for **Joyería y Relojería Cuore**. This site reflects 40 years of tradition, craftsmanship, and technical precision.

## Strategic Identity
- **Legacy-Driven Design**: Editorial layout focusing on the brand's history and artisan workshop.
- **Precision Engineering**: Visual markers for technical details (±0.05mm) and high-authority metadata.
- **Consultation CRO**: Direct-to-WhatsApp consultation funnel to maintain exclusivity.

## Tech Stack
- **Engine**: React + TanStack Start (SSR/Hydration).
- **Styling**: Tailwind CSS (Bespoke design tokens).
- **Motion**: Intersection Observer Reveals (Bespoke coordination).
- **Typography**: Lora (Editorial) & Arimo (Technical).

## Getting Started
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
